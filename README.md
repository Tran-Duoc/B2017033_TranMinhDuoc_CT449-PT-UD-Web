### Bài tập thực hành phát triển ứng dụng web (CT449).

<hr>

**- dowload code tại đây**

```shell
git clone https://github.com/Tran-Duoc/CT449-PT-UD-Web.git
```

**- Đối với file backend**

```shell
  cd backend
  cd src
  npm install
```

**- Đối với file frontend**

```shell
  cd frontend
  cd src
  npm install
```

**- Chạy dự án**
_+ Bên file backend_

```shell
cd src
npm start
```

_- Bên file frontend_

```shell
  cd src
  npm run dev
```
